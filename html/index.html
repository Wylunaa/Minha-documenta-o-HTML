Paragráfos e quebra de linhas

Títulos:

Você pode escrever qualquer título de qualquer assunto que seja de acordo com o assunto do seu site. Tudo que você colocar entre as tag <h1> e </h1> vira um titulo.

Paragráfos:

Você pode escrever um Parágrafo de qualquer assunto que seja de acordo com o assunto do seu site.Tudo que você colocar entre as tags < p> e </p> vira um Parágrafo. Para os elementos do seu paragráfos não ficarem tudo juntos você deve usar a tag <br> ou para ficar mais organizado você pode dividir o conteúdo em section usando a tag <section> e </section>.

Símbolos especias:

como colocar o símbolo "<" e ">"

O significado da sigla para colocar o símbolo maior que ( < ) e menor que ( > ) vem do inglês Less than para o símbolo maior que e para o menor que Greate than por isso tem a sigla "lt" para o maior que e "gt" para o menor que.
Para colocar o símbolo de maior que (" < ") você deve usar o "e" comecial & + lt + ;
Para colocar o símbolo de menor que (" > ") Você deve usar o "e" comercial & + gt +;
Para colocar o símbolo do e comercial (" & ") Você deve usar o "e" comercial & + amp + ;
